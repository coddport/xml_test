Колонки в конфиге должны быть вот такими (разделитель - '\t'):
on	test_id	format	file_name	flag comment



on
	1 = генеровать тест
	0 = не генерировать тест
test_id
	название теста, должно быть уникальным между всеми файлами
format (пока не подключала, по умолчанию работает как s)
	a - проверка реплики бота
	c - проверка класса ответа (до go)
	s - проверка состояния ответа (после go)
file_name
	название файла, должно быть в той же папке, что и скрипт
flag
	что угодно
comment
	что угодно



Колонки в файлах диалогов должны быть (разделитель - '\t'):
test_id	line	q	a_state	a_class	a	flag

test_id
	название теста
line
	счетчик реплик (начинается с 0)
q
	реплика юзера
a_state
	стейт цели (то есть после перехода по go!)


a_class (не подключено)
	стейт цели (то есть после перехода по go!)
a	(не подключено)
	реплика бота
flag
	что угодно
